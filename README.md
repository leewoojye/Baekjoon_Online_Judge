# Baekjoon Online Judge

This repository contains problem-solving code for **Baekjoon Online Judge (BOJ)**, along with some lecture-based data structure practice files.

## Overview

- Most source files are named by BOJ problem number.
- The repository includes solutions written mainly in **C++**, with some **C** and **Python** files.
- The `자료구조/` directory contains code organized from data structure lecture PDFs and exercises.

## Structure

- `*.cpp`: BOJ C++ solutions
- `*.c`: C practice files and lecture examples
- `*.py`: Python practice files
- `자료구조/`: data structure examples, quizzes, and notes organized by week

## How To Run

### C++

```bash
g++ -std=c++17 -O2 -Wall -Wextra -o main 1000.cpp
./main
```

### C

```bash
cc -std=c11 -O2 -Wall -Wextra -o main example.c
./main
```

### Python

```bash
python3 script.py
```

## Notes

- This repository does not use a single build system.
- Each file is intended to be compiled or executed individually.
- Some folders and duplicate filenames reflect personal practice history.
